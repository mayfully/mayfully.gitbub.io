Flitto

APP UX / UI Design

플리토는 집단지성을 활용한 글로벌 번역 플랫폼이다. 사용자들이 자유롭게 번역요청을 하고 번역을 받을 수 있는 공간이다. 요청자로서 번역 요청을 할 수 있고, 외국어를 할 수 있다면 직접 번역가가 될 수도 있다. 플리토는 18개의 다국어를 지원하므로 UX 및 UI 디자인을 할 때 고려해야 하는 부분들이 많았다. 예를 들어 영어를 비롯해 인도네시아어, 러시아어 등 길이가 긴 언어들을 화면에 배치해야 하는 상황이라면 직렬보다는 병렬로, 언어가 길어지면 2줄 이상으로 최대 기준점을 고려해 디자인하였다. 보다 제한적인 상황들이 많았지만 이번에 플리토를 새롭게 디자인할 때 요청자와 번역가 두 모드로 나누었고 각 모드에 맞춰 카드디자인, 요청하기, 번역하기 세부화면 디자인 등 플랫폼의 UI를 발전해 나갔다. 플리토 대표 메인컬러는 집단지성 요청하기 모드, 짙은 회색은 더욱 전문적이고 고급스러움을 살리기 위해 번역가 모드의 색상으로 지정하였다.

App UX / UI Design

해외여행을 가서 외국인에게 의사전달을 해야 하는 상황이라면, 빠른 기계번역도 좋지만 빠르고 정확하게 사람이 해주는 번역을 택할 것이다. 플리토에선 데이터를 기반으로 실시간으로 번역을 제공해주는 기능을 새로 추가하였고, 사용자가 번역요청시 아래 유사한 번역을 추천하여 보여주는 방식이다. 사용자가 비교할 수 있는 원문을 더 돋보이게 크기와 색상으로 번역문과 구별하였다. 하단에 번역문, 번역 듣기 그리고 복사할 수 있도록 사용자 경험을 높였다.
